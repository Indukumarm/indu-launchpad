# Guidewire Deployment via Kubernetes

## Architecture

Guidewire InsuranceSuite (PolicyCenter, BillingCenter, ClaimCenter) on Kubernetes.

## Helm Deployment

```yaml
# values.yaml
replicaCount: 3
image:
  repository: guidewire/policycenter
  tag: "10.2.0"
resources:
  requests:
    memory: "8Gi"
    cpu: "4"
```

```bash
helm install guidewire ./guidewire-chart -f values.yaml
```

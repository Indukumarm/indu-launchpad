# Redgate Database Release Automation

## Database Release Flow

<Diagram>{`
flowchart LR
    A[Dev Database] --> B[Schema Compare]
    B --> C[Generate Script]
    C --> D[Review Changes]
    D --> E[Deploy to Test]
    E --> F[Validation]
    F --> G[Deploy to Prod]
    G --> H[Rollback Ready]
`}</Diagram>

## Schema Comparison

```powershell
# Compare schemas
SQLCompare /server1:prod-db /db1:myapp /server2:staging-db /db2:myapp /report:diff.html
```

## CI/CD Integration

```yaml
- name: Deploy Database Changes
  run: |
    sqlpackage /Action:Publish \
      /SourceFile:myapp.dacpac \
      /TargetServerName:prod-sql \
      /TargetDatabaseName:myapp
```

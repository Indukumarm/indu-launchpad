# Linux for DevOps Engineers

## Essential Commands

```bash
# File operations
ls -lah          # List files with permissions
cd /var/log      # Change directory
cat app.log      # View file
tail -f app.log  # Follow log file

# Process management
ps aux | grep nginx
kill -9 PID
systemctl status nginx
systemctl restart nginx

# Networking
netstat -tulpn   # List open ports
curl https://api.example.com
wget https://example.com/file.tar.gz

# Disk usage
df -h            # Disk space
du -sh /var/log  # Directory size
```

## Shell Scripting

```bash
#!/bin/bash
for service in nginx postgres redis; do
  systemctl status $service || systemctl start $service
done
```

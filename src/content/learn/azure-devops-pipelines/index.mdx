# Azure DevOps Pipelines — Multistage Releases

## Multistage YAML Pipeline

```yaml
trigger:
  branches:
    include:
      - main

stages:
  - stage: Build
    jobs:
      - job: BuildJob
        pool:
          vmImage: 'ubuntu-latest'
        steps:
          - task: Maven@3
            inputs:
              goals: 'clean package'
          - task: PublishBuildArtifacts@1

  - stage: DeployDev
    dependsOn: Build
    jobs:
      - deployment: DeployDevJob
        environment: dev
        strategy:
          runOnce:
            deploy:
              steps:
                - task: AzureWebApp@1

  - stage: DeployProd
    dependsOn: DeployDev
    jobs:
      - deployment: DeployProdJob
        environment: production
        strategy:
          runOnce:
            deploy:
              steps:
                - task: AzureWebApp@1
```

## Approval Gates

Configure manual approvals in Environments → Approvals and checks.
